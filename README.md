# Curso de Programación Python - Talento Tech 🐍🚀

Este repositorio está dedicado a almacenar todos los proyectos, desafíos de código, algoritmos y prácticas desarrolladas a lo largo de mi formación en el programa **Talento Tech**. 

El objetivo de este espacio es documentar mi proceso de aprendizaje, la evolución de mi lógica de programación y la aplicación de buenas prácticas en el desarrollo backend con Python.

---

### 📂 Estructura del Repositorio

Para mantener el código limpio y navegable, el contenido se organiza de la siguiente manera:

* `Proyectos/`: Proyectos integradores y entregables principales (como el Sistema de Gestión de Inventario).
* `Desafios/`: Retos de código, algoritmos y ejercicios prácticos de lógica.
* `Notas/`: Apuntes y scripts de prueba de las clases diarias.

---

### 🛠️ Tecnologías y Conceptos Practicados

* **Lenguaje principal:** Python 3.x
* **Estructuras de Control:** Bucles (`while`, `for`), Condicionales anidados (`if-elif-else`).
* **Manejo de Datos:** Listas dinámicas, diccionarios y manipulación de strings.
* **Buenas Prácticas:** Programación defensiva (validación de datos de usuario), modularización y limpieza de código.
